# SAP-ABAP-Customer-Due-Reminder-Report
1. Title
Customer Due Reminder Report — SAP ABAP ALV Report with Overdue Highlighting

2. Description (for GitHub repo)
A custom SAP ABAP ALV Report that fetches customer payment data from a transparent table ZSTOCKDUE and highlights overdue accounts (>30 days) in red for quick finance team action.

3. Features to list in README

Custom ALV report built using REUSE_ALV_GRID_DISPLAY
Data sourced from custom transparent table ZSTOCKDUE
Automatic red-row highlighting for invoices overdue by more than 30 days using INFO_FIELDNAME
ALV layout customization via SLIS_LAYOUT_ALV
Clean field catalog built using SLIS_FIELDCAT_ALV
Supports finance teams in sending targeted payment reminders
Lightweight and easily extensible for additional business logic
